How It's Made

The front end is an Angular app that is being served by GitHub pages.

The quotes are fetched from a Node.js/Express REST API that I created and is hosted by Heroku at https://ram-dass-quotes-api.herokuapp.com/. The GitHub pages for the Angular application and for the REST API can be found HERE and HERE, respectively.

If you would like to contribute a quote to the collection, please submit a pull request to the REST API with the quote's category, text, and the source of it.
